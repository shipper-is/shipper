# Shipper marketing site

Next.js landing page for the Shipper CLI.

## Local development

From this directory:

```sh
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

Set the project **Root Directory** to `web/` in the Vercel dashboard. The default Next.js build works as-is; no `vercel.json` is required.
